# **Fractol** #

### How it work? ###

* Run Makefile.
* ./fractol [name_fract] (Example: ./fractol Julia).

### Features ###

* Change a coefficient (if it's Julia).
* Zomming in dot.
* Change color scheme.
* Move fractol (left, right, up, down).
* Change number of iteration and radius.
* Change fractol (Press: 1...8)
